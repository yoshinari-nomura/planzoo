#+TITLE: planzoo - 学年暦と時間割から MHC エントリを生成
#+AUTHOR: Yoshinari Nomura
#+OPTIONS: toc:2

* 概要

planzoo は，大学の学年暦と時間割データから [[https://github.com/yoshinari-nomura/mhc][MHC (Message Harmonized Calendar)
]] 形式のカレンダーエントリを自動生成する Ruby スクリプトです．

以下の機能を持ちます:
+ 講義スケジュールを繰り返し予定として登録
+ 曜日読換 (祝日等で別曜日の授業を行う場合) を自動考慮
+ 休講日を自動的に除外
+ 大学イベント (入学式，卒業式等) の登録

* 必要環境

+ Ruby
+ MHC がインストールされていること (=~/MHC/draft= ディレクトリが存在すること)

* 使い方
** 基本的な使い方

#+begin_src bash
./planzoo timetable-example.org
#+end_src

MHC エントリが =~/MHC/draft= ディレクトリに生成されます．

** デバッグモード

=--debug= オプションを付けると，ファイルを作成せずに標準出力に MHC 形式で表示します．

#+begin_src bash
./planzoo --debug ./timetable-example.org
#+end_src

* 入力ファイル形式

入力ファイルは Org-mode のテーブル形式で記述します．以下のシートが必要です．

** Lecture (講義)

| term     | 学期 (1, 2, …)                              |

** Period  (時限)

各時限の開始・終了時刻．カレンダに登録されない

| カラム名 | 説明           |
|----------+----------------|
| nth      | 時限番号       |
| s_time   | 開始時刻       |
| e_time   | 終了時刻       |

例:
| nth | s_time | e_time |
|-----+--------+--------|
|   1 |  08:40 |  09:30 |
|   2 |  09:40 |  10:30 |
|   3 |  10:45 |  11:35 |
|   4 |  11:45 |  12:35 |
|   5 |  13:25 |  14:15 |
|   6 |  14:25 |  15:15 |
|   7 |  15:30 |  16:20 |
|   8 |  16:30 |  17:20 |
|   9 |  17:30 |  18:20 |
|  10 |  18:30 |  19:20 |

** Term (学期)

+ 各学期の Lecture を配置すべき期間 (講義開始日と終了日)
+ カレンダに登録されない
+ 試験期間を含めるかどうかは，試験をカレンダー上に配置したいかに依存する．

| カラム名 | 説明                  |
|----------+-----------------------|
| nth      | 学期番号 (1-4)        |
| s_date   | 開始日(YYYYMMDD 形式) |
| e_date   | 終了日(YYYYMMDD 形式) |

例:
| nth |   s_date |   e_date |
|-----+----------+----------|
|   1 | 20260408 | 20260609 |
|   2 | 20260610 | 20260806 |
|   3 | 20261001 | 20261203 |
|   4 | 20261204 | 20270212 |

** DeemedWeek (曜日読換)

+ 講義時間数調整のための曜日読換表
+ カレンダに登録される
+ Lecture 配置の際に考慮される

| カラム名    | 説明                     |
|-------------+--------------------------|
| date        | 対象日 (YYYYMMDD 形式)   |
| deemed_week | みなし曜日 (Mon, Tue 等) |
| summary     | カレンダー上の表示名     |

例:
|     date | deemed_week | summary      |
|----------+-------------+--------------|
| 20260507 | Tue         | 火曜日の授業 |
| 20260508 | Wed         | 水曜日の授業 |
| 20260722 | Mon         | 月曜日の授業 |
| 20261104 | Mon         | 月曜日の授業 |
| 20261119 | Mon         | 月曜日の授業 |
| 20270112 | Mon         | 月曜日の授業 |
| 20270113 | Fri         | 金曜日の授業 |
| 20270205 | Thu         | 木曜日の授業 |

* 出力

以下の MHC エントリが =~/MHC/draft= ディレクトリに生成されます:

1. *講義エントリ* - 各講義の繰り返し予定(曜日読換・休講日を考慮した例外日付き)
2. *曜日読換エントリ* - 曜日読換日のリマインダー
3. *イベントエントリ* - 各種大学イベント
